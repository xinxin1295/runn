# 润 · Runn

给大厂痛风PM/程序员的痛风友好工作陪伴者。

信信 · AI实战课第一节作业

## 〇 两种形态

- **Web 版** [润.html](润.html)：单 HTML 双击即跑，作业提交版本，无需安装
- **桌面 .exe 版** [desktop/](desktop/)：Electron 打的 Windows 桌面应用，托盘常驻+右下角迷你卡片+开机自启，对齐一线大厂桌面软件体验（详见 [desktop/README-桌面版.md](desktop/README-桌面版.md)）

两个形态共享同一套设计哲学、同一套内容库、同一套关节镇预测机制，桌面版是 Web 版的**升级形态**，不是替代。

## 一 三十秒看懂

**它不是喝水APP，它是一个不会被关掉的陪伴者。**

在大厂坐一整天，日程被会议塞满，打开就忘了关，忘了关就一直陪你，陪你到你痛过又不想再痛的那种真的能坚持下来。核心不是记录你喝了多少，是让你看到不喝的代价：主界面永远显示「距下次可能急性发作 X 天」，喝一杯这个数字就往后推一天。

## 二 怎么打开

**方法一：直接双击**

在文件夹里双击 `润.html`，Chrome/Edge/Firefox 都能跑。第一次打开会有一片汝窑天青色的静水，中间是水盂，你在电脑前面工作它就在那里。

**方法二：设为浏览器书签**

打开 `润.html` 后，Ctrl+D 加书签，钉在书签栏，之后一秒能唤出。

**方法三：设为开机自启（推荐）**

这才是它真正的样子——每天开机它自动出现，你不用主动想起。产品里内置了引导：点右下角「开机自启引导」按钮，三步搞定。也可以照下面做：

1. 按 `Win + R`，输入 `shell:startup`，回车
2. 右键 `润.html`，选择「发送到 → 桌面快捷方式」
3. 把桌面上生成的快捷方式拖进第 1 步打开的启动文件夹

从此开机就打开。

## 三 怎么用

- **顶部三态**：会议中 / 深度专注 / 空隙时刻。会议中一切通知静默，只有屏幕边缘微弱的呼吸感光晕；深度专注不弹窗但保持存在；空隙时刻允许桌面通知
- **中央水盂**：今日饮水量。水盂里的水位随你打卡实时上涨，有慢波纹动画
- **打卡三档**：一小口 125ml / 一整杯 250ml / 大杯 500ml，点了就记下
- **观察语**：每次打卡或定时切换一句话，不会命令你、不会催你，只是观察式提醒
- **关节镇**：底部关节图 + 距下次急性发作预测天数 + 连续达标天数
- **近七日**：过去七天的饮水柱状回顾，达标日青色，未达标日淡灰带一颗红点
- **睡眠模式**：0-6 点自动进入夜色，不打扰
- **右上角 ×**：关闭按钮点下去会出现挽留对话框，问你要不要再留一天，二次确认才关

## 四 为什么要这么设计

这版跟师兄课上那版最大的差异，是用户不同。

师兄那版：自由职业者，日程自己安排，喝水提醒响就响。
这版：大厂 PM / 程序员，一天六场会，Slack 常青灯，一个不合时宜的提醒会让他直接卸载。

推导过程（用课上的福格模型）：

| 要素 | 大厂痛风人的现状 |
|---|---|
| 动机 M | 高（怕再痛，也在吃药） |
| 能力 A | 高（工位有水杯） |
| 触发 T | 极低（会议连轴、注意力锁死） |

跟师兄一样，缺的是触发。但触发方案不能照抄——**大厂人对"错时机的触发"完全零容忍**。

对症下药三招：

1. **状态感知**：三态切换，会议中静默一切
2. **代价可见**：不显示喝了多少，显示下一次痛风大概什么时候（这才是留住痛风人的钩子）
3. **关闭挽留**：把关闭动作从下意识变成有意识，让离开的成本从"点一下 ×"变成"要读完一段挽留然后再点"

## 五 用课上的定位反查验收

课上说：**用定位去验收**。定位是「给大厂痛风PM/程序员的痛风友好工作陪伴者」。

- [x] 会议中静音（差异化核心）：切到会议中，所有通知禁用，光晕熄灭
- [x] 无命令式话术：全部观察+邀请式，无「该喝水了」这种话
- [x] 关闭有情感成本：右上角 × 二次确认
- [x] 显示不喝的代价：关节镇「距下次可能急性发作 X 天」永远在
- [x] 无 AI 爆款审美：无紫渐变、无 Inter，只有汝窑天青+思源宋体+Fraunces
- [x] 双击可跑：无框架依赖，单 HTML
- [x] 开机自启方案完整：产品内置引导，README 也写了

## 六 技术栈

- 单 HTML 文件，无框架、无构建、无依赖
- Google Fonts CDN 加载思源宋体+Fraunces
- localStorage 存所有数据，不上传
- Notification API 桌面通知
- Wake Lock API 保持屏幕活跃（可选）
- Page Visibility API 检测切换回来时刷新
- 所有配色 CSS 变量集中定义，方便微调
