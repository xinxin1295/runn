## 这是什么

一款常驻电脑的桌面软件,专门给**一天开六场会、痛风缠身的大厂程序员和产品经理**。

它不记你喝了多少水,它让你看见**不喝的代价**:主界面永远显示「按当前规律,距下次可能急性发作还有几天」。喝一杯,这个数字往后推;断卡,它就缩短。

## 定位是怎么推导出来的(这节课的方法)

用福格行为模型 `行为 = 动机 × 能力 × 触发` 给「大厂痛风人喝水」这件事做诊断:

| 要素 | 现状 | 结论 |
|---|---|---|
| 动机 | 高(痛过一次,怕再痛) | 不缺 |
| 能力 | 高(工位就有水杯) | 不缺 |
| **触发** | **极低(会议连轴、注意力锁死)** | **缺的是这个** |

和师兄课上那版一样,缺的是触发。但大厂人有个师兄没提的额外约束:**对错时机的触发零容忍**,会议里一响就直接卸载。所以定位不是「喝水提醒」,而是「**痛风友好的工作陪伴者**」,三个差异化设计全从这来:

- **会议中静默**:开会前切一下,所有提醒闭嘴,不在评审会上社死
- **发作预报**:不显示喝了多少,显示距下次可能急性发作还有几天,让代价可见
- **关闭挽留**:点关闭要二次确认,让离开从下意识变成有意识

## 功能

- 水盂随饮水量实时上涨,水波呼吸动画
- 三档打卡(125 / 250 / 500 ml),打错可撤销
- 三态切换:会议中(静默) / 深度专注(轻提醒) / 空隙时刻(立即提醒)
- 会议静默到点自动回到深度专注,忘了切回来也不会一直哑着
- 发作预报 + 连续达标天数 + 近七日饮水回顾
- 每日目标可调(痛风人默认 2500ml,肾功能弱可调低)
- 桌面版:系统托盘常驻、右下角迷你卡片弹窗、开机自启、关闭到托盘不真退出
- 新手引导:首次打开逐步带你认识每个功能
- 全程观察式话术,不命令你、不催你

## 下载与安装

**Windows**:到 [Releases](https://github.com/xinxin1295/runn/releases/latest) 下载 `润Runn_Setup_1.0.0.exe`,双击安装。首次运行自动加开机自启。Windows Defender 若拦截,点「更多信息 → 仍要运行」放行(个人开发无签名的正常现象)。

**Mac**:到 [Releases](https://github.com/xinxin1295/runn/releases/latest) 下载 `润Runn_1.0.0_mac.dmg`,双击打开,把「润 Runn」拖进「应用程序」文件夹。首次打开:

1. 在「应用程序」里**右键点「润 Runn」→ 选「打开」→ 弹窗里再点「打开」**(不要直接双击,直接双击会被拦)。
2. 如果是苹果芯片(M1/M2/M3)的 Mac,系统可能提示装 Rosetta,点「安装」等一下即可(一次性,本程序用 x64 架构,靠 Rosetta 在苹果芯片上运行)。
3. 万一仍提示「已损坏」,打开「终端」粘这行回车再打开:`xattr -dr com.apple.quarantine "/Applications/润 Runn.app"`

这套流程对所有没买苹果付费签名的个人应用都一样,不是程序本身的问题。

## 数据存哪

全部存本地,每个用户各存各的,不上传任何服务器:

- Windows:`%APPDATA%\润 Runn\runn-state.json`
- Mac:`~/Library/Application Support/润 Runn/runn-state.json`

删了这个文件就清零,换电脑不同步。
